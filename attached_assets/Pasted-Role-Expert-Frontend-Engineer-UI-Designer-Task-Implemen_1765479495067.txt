Role: Expert Frontend Engineer & UI Designer
Task: Implement a high-performance, animated Strategic AI Dashboard for BlueAlly. This dashboard must be dynamic, accepting a JSON data object to render reports for different clients.

1. Project Setup & Dependencies

Please install the following dependencies immediately:
npm install framer-motion recharts lucide-react

2. Implementation Strategy

Create a new component file (e.g., src/components/Dashboard.jsx or .tsx) and implement the code below.

Key Features to Maintain:

Dynamic Data Injection: The Dashboard component accepts a data prop. If no prop is provided, it falls back to DEFAULT_DATA (currently set for Synovus Bank).

Animations: Use framer-motion for entrance animations (fade-ins, slide-ups) and the "Flywheel" background rotation.

Interactive Charts: Use recharts for the Priority Matrix (Bubble/Scatter Chart).

Icon Mapping: The ICON_MAP object allows JSON data to reference icons by string name (e.g., "TrendingUp"), making the data portable.

3. The Code

Paste this complete code into the dashboard component file:

const AnimatedCounter = ({ value, prefix = "", suffix = "" }) => {
  const [displayValue, setDisplayValue] = useState(0);
  
  useEffect(() => {
    let start = 0;
    const end = parseFloat(value);
    const duration = 2000;
    const increment = end / (duration / 16);

    const timer = setInterval(() => {
      start += increment;
      if (start >= end) {
        setDisplayValue(end);
        clearInterval(timer);
      } else {
        setDisplayValue(start);
      }
    }, 16);

    return () => clearInterval(timer);
  }, [value]);

  return (
    <span>{prefix}{displayValue.toFixed(1)}{suffix}</span>
  );
};
